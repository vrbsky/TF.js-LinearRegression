In-Browser Linear Regression Prediction with TensorFlow.js

This page performs a both training and inference on client side (i.e. directly in your browser) of a Linear Regression model.
It page learns a simple machine learning model which can then be used to predict a correct output based on the learning data.
The training values used for the model are according to a fuction y = -x. 40 data samples on the interval [-10, 10] are used for training.
Therefore, an input of x = 5 should generate an output y = -5.
